# Pioneer AVR LX83 Integration

Control your Pioneer AVR LX83 receiver from Home Assistant via Telnet.

## Features

- Power control
- Volume control
- Source selection
- Mute control
- Real-time status updates

## Configuration

Add the integration via the UI:
1. Go to Settings > Devices & Services
2. Click "+ Add Integration"
3. Search for "Pioneer AVR LX83"
4. Enter your device details
